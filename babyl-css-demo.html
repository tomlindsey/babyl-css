For any width that is a multiple of 60

Red items are multiples of 60 and may be subdivided using any of the ways outlined below -- there are might be others, I did not try to make an exhaustive list; I am just playing around.

Example: Let's say a page is 960px wide so the body's style is set using style: b960_c1. The developer would like this page divided into equal regions, each 480 pixels wide so applies style: b960_c2 to the page elements. The developer could then carve the 480 pixel spaces up by using any of the styles: b480_c*
